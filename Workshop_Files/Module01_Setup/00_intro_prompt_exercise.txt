# แบบฝึกหัดที่ 1: Prompt แรกของคุณ

ลองพิมพ์ Prompt ต่อไปนี้ใน Claude Cowork แล้วดูผลลัพธ์

---

PROMPT 1 (ง่าย):
"สวัสดีครับ/ค่ะ ฉันชื่อ [ชื่อของคุณ] ทำงานตำแหน่ง [ตำแหน่ง] บริษัท [ชื่อบริษัท] 
ช่วยแนะนำว่าคุณจะช่วยงานฉันได้อย่างไรบ้างในชีวิตประจำวัน"

---

PROMPT 2 (ทดสอบ File Access):
"ช่วยดูไฟล์ทั้งหมดในโฟลเดอร์ workshop ให้หน่อย แล้วบอกว่ามีไฟล์อะไรบ้าง"

---

PROMPT 3 (ทดสอบ Multi-step):
"ฉันต้องการ:
1. รายชื่อไฟล์ทั้งหมดในโฟลเดอร์
2. จัดกลุ่มตามประเภทไฟล์
3. สรุปเป็นตาราง

ช่วยดำเนินการตามลำดับนี้"

---

สังเกตความแตกต่างของผลลัพธ์แต่ละ Prompt
