# ✅ Checklist ก่อนเริ่ม Workshop

## Claude Account
- [ ] สมัครสมาชิก Claude Pro หรือ Max แล้ว
- [ ] Login เข้าสู่ระบบสำเร็จ

## Claude Desktop App
- [ ] ดาวน์โหลด Claude Desktop App เวอร์ชันล่าสุด
- [ ] ติดตั้งสำเร็จบน macOS / Windows
- [ ] เปิดแอปได้ปกติ

## Cowork Mode
- [ ] เปิดใช้งาน Cowork Mode ในการตั้งค่า
- [ ] เลือก Working Directory (โฟลเดอร์ทำงาน) แล้ว
- [ ] ทดสอบ: พิมพ์ "สวัสดี ช่วยอ่านไฟล์ในโฟลเดอร์นี้ให้หน่อย"

## ไฟล์ฝึก
- [ ] ดาวน์โหลดชุดไฟล์ฝึก 83 ไฟล์แล้ว
- [ ] วางไว้ในโฟลเดอร์ที่ Claude สามารถเข้าถึงได้

## อุปกรณ์
- [ ] แบตเตอรี่เพียงพอ / ชาร์จสายพร้อม
- [ ] เชื่อมต่อ WiFi แล้ว
- [ ] หูฟัง (ถ้าต้องการ)
