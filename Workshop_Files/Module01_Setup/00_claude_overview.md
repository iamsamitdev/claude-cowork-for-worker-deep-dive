# Claude Cowork — ภาพรวมสำหรับผู้เริ่มต้น

## Claude Cowork คืออะไร?

Claude Cowork เป็น **Agentic AI Mode** ของ Claude Desktop App
ที่เปลี่ยน Claude จากแชทบอทธรรมดา ให้กลายเป็น AI ที่สามารถ:

- 📁 **อ่านและเขียนไฟล์** ในเครื่องของคุณได้โดยตรง
- 🔧 **รันโค้ด** เพื่อประมวลผลข้อมูล
- 🌐 **เชื่อมต่อเครื่องมือ** เช่น Google Drive, Gmail, Slack
- ⏰ **ตั้งงานอัตโนมัติ** ด้วย Scheduled Tasks
- 📊 **สร้างเอกสาร** Excel, Word, PowerPoint

## ต่างจาก Claude.ai (เว็บ) อย่างไร?

| Claude.ai (เว็บ) | Claude Cowork |
|-----------------|---------------|
| ตอบคำถาม ช่วยเขียน | ทำงานจริงหลายขั้นตอน |
| ไม่เข้าถึงไฟล์ | อ่าน/เขียนไฟล์ได้ |
| Session เดี่ยว | จำ Context ต่อเนื่อง |
| ไม่มี Tools | มี Skills & Connectors |
| Manual ทุกอย่าง | Automate ได้ |

## เริ่มต้นใช้งาน

1. เปิด Claude Desktop App
2. คลิก "Cowork" หรือเปิด Cowork Mode
3. เลือกโฟลเดอร์ทำงาน
4. เริ่มพิมพ์งานที่ต้องการ

## Tips สำหรับมือใหม่

✅ บอก Context ว่าคุณเป็นใคร ทำงานอะไร
✅ ระบุ Output ที่ต้องการชัดเจน (ไฟล์, format)
✅ แบ่งงานใหญ่เป็น Step ย่อย
✅ ตรวจสอบผลลัพธ์ก่อน confirm
