# แบบฝึกหัด: จัดระเบียบ Workspace

## โจทย์
โฟลเดอร์ `messy_files/` มีไฟล์ 10 ไฟล์ที่ไม่ได้จัดระเบียบ
ให้ใช้ Claude Cowork จัดการโดยใช้ Prompt ต่อไปนี้:

---

### Prompt ที่ 1: สำรวจก่อน
```
อ่านและสรุปเนื้อหาของทุกไฟล์ในโฟลเดอร์ messy_files/
บอกฉันว่าแต่ละไฟล์มีเนื้อหาเกี่ยวกับอะไร
```

### Prompt ที่ 2: เสนอโครงสร้าง
```
จากไฟล์ทั้งหมดใน messy_files/ ให้เสนอโครงสร้างโฟลเดอร์ใหม่
ที่เหมาะสมสำหรับองค์กร พร้อมอธิบายเหตุผล
```

### Prompt ที่ 3: สร้าง Index
```
สร้างไฟล์ index.md ที่รวม:
- รายชื่อไฟล์ทั้งหมด
- หมวดหมู่
- คำอธิบายสั้นๆ ของแต่ละไฟล์
- วันที่แนะนำให้ Review
```

### Prompt ที่ 4: ตั้งชื่อใหม่
```
เสนอชื่อไฟล์ใหม่ที่มี convention ชัดเจน 
รูปแบบ: YYYY-MM_category_description.ext
```

---

- [ ] ชื่อไฟล์ใหม่ follow convention
