# โครงการ: ระบบ CRM ใหม่

## ภาพรวม
โครงการพัฒนาระบบบริหารลูกค้าสัมพันธ์ (CRM) เพื่อรองรับการขยายธุรกิจ
ในปีงบประมาณ 2569

## วัตถุประสงค์
1. รวมข้อมูลลูกค้าจากทุกแผนกในที่เดียว
2. ติดตาม Sales Pipeline แบบ Real-time
3. ลดเวลา Data Entry ลง 60%

## ทีมงาน
- Project Manager: คุณวิภา สมบูรณ์
- Tech Lead: คุณธนา ใจดี
- Business Analyst: คุณพิม รักงาน

## Timeline
- Phase 1 (เม.ย. - พ.ค.): Requirements & Design
- Phase 2 (มิ.ย. - ส.ค.): Development
- Phase 3 (ก.ย.): Testing & UAT
- Phase 4 (ต.ค.): Go-Live
