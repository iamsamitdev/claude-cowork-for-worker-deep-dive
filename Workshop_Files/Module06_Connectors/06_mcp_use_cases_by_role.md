# MCP Use Cases สำหรับคนทำงาน

## กลุ่มที่ 1: Morning Workflow (เริ่มวันใหม่อย่างมีประสิทธิภาพ)

### 🌅 Daily Briefing (ต้องการ: Gmail + Calendar + Slack)

```
"สวัสดีตอนเช้า ช่วยสรุปให้หน่อยว่า:
1. อีเมลสำคัญที่ยังไม่ได้ตอบจากเมื่อวาน
2. ตารางประชุมวันนี้พร้อมเวลา
3. ข้อความใน Slack #general ที่ tag ฉันไว้
และสรุปว่าควรเริ่มจากเรื่องไหนก่อน"
```

```
"วันนี้มีประชุมอะไรบ้าง? แต่ละนัดมีเอกสารเตรียมการใน Drive ไหม
ถ้ามี ช่วยสรุปประเด็นสำคัญที่ควรรู้ก่อนเข้าห้องประชุม"
```

```
"ดูอีเมลตั้งแต่เมื่อคืนถึงตอนนี้ แล้วแยกเป็น 3 กลุ่ม:
🔴 ด่วน — ต้องตอบภายในวันนี้
🟡 ปกติ — ตอบภายในพรุ่งนี้ได้
⚪ FYI — อ่านแล้วก็พอ"
```

### 📋 Weekly Planning (ต้องการ: Calendar + Drive)

```
"ดูตาราง Meeting สัปดาห์นี้ แล้วช่วยหาช่วงเวลาว่าง 2 ชั่วโมง
ที่เหมาะสำหรับงาน Deep Work ที่ไม่มีการรบกวน"
```

```
"สรุป OKR ของทีมจากไฟล์ใน Drive ชื่อ 'Q2_OKR.docx'
แล้วเช็คว่า Meeting สัปดาห์นี้มีอันไหนเกี่ยวข้องกับ OKR บ้าง"
```

## กลุ่มที่ 2: Email Management (จัดการอีเมลอย่างชาญฉลาด)

### 📧 อ่านและสรุปอีเมล (ต้องการ: Gmail)

```
"สรุปอีเมลจาก [ชื่อลูกค้า] ทั้งหมดในสัปดาห์นี้
เน้นประเด็น: สิ่งที่เขาร้องขอ, deadline ที่ระบุ, และสิ่งที่ยังรอคำตอบ"
```

```
"ค้นหาอีเมลทุกฉบับที่มีคำว่า 'invoice' หรือ 'ใบแจ้งหนี้' ในเดือนนี้
สรุปว่ามียอดรวมเท่าไร และมีฉบับไหนที่ยังค้างชำระ"
```

```
"อีเมลไหนบ้างที่ฉันส่งไปแต่ยังไม่มีคนตอบกลับเกิน 3 วัน
ช่วยร่าง Follow-up Email ให้แต่ละฉบับแบบสุภาพ"
```

### ✍️ ร่างอีเมล (ต้องการ: Gmail + Drive)

```
"อ่านอีเมลล่าสุดจาก [ชื่อ] แล้วช่วยร่างอีเมลตอบกลับ
โดยใช้ข้อมูลจากไฟล์ 'Project_Status.xlsx' ใน Drive ประกอบด้วย
ใช้ภาษาทางการ กระชับ ไม่เกิน 200 คำ"
```

```
"ร่างอีเมลแจ้งผลการประชุมวันนี้ให้ทีมทุกคน
โดยสรุปจาก Meeting Notes ในไฟล์ Drive ชื่อ 'Meeting_[วันที่].docx'
ระบุ Action Items และ Deadline ให้ชัดเจน"
```

## กลุ่มที่ 3: Document Management (จัดการเอกสารใน Drive)

### 📁 ค้นหาและสรุปเอกสาร (ต้องการ: Google Drive)

```
"หา Proposal ทั้งหมดที่อยู่ใน Drive ของฉัน
เรียงตาม Modified Date ล่าสุด แล้วสรุปว่าแต่ละไฟล์เกี่ยวกับอะไร"
```

```
"อ่านไฟล์ 'Company_Policy_2025.pdf' แล้วสรุปนโยบายที่เกี่ยวกับ
การลา, การทำงานจากบ้าน, และค่าใช้จ่ายในการเดินทาง"
```

```
"เปรียบเทียบ Proposal สองไฟล์ใน Drive:
'Proposal_v1.docx' กับ 'Proposal_v2.docx'
บอกว่าอะไรเปลี่ยนไปบ้าง และเวอร์ชันไหนสมบูรณ์กว่า"
```

### 📊 วิเคราะห์ข้อมูลจาก Spreadsheet (ต้องการ: Drive)

```
"เปิดไฟล์ 'Sales_Q1.xlsx' ใน Drive
สรุปยอดขายรายสินค้า Top 5, สินค้าที่ยอดขายตกมากที่สุด
และแนวโน้มเมื่อเทียบกับ Q1 ปีที่แล้ว"
```

```
"ดูไฟล์ Budget ล่าสุดใน Drive แล้วบอกว่า
Category ไหนใช้เงินเกิน Budget ไปบ้าง และมากแค่ไหน"
```

## กลุ่มที่ 4: Calendar & Meeting Management (นัดหมายและประชุม)

### 📅 จัดการตาราง (ต้องการ: Google Calendar)

```
"ช่วยหาเวลาว่างของฉันในสัปดาห์หน้า (จ-ศ)
ที่เหมาะสำหรับประชุม 1 ชั่วโมง ระหว่าง 10:00-16:00
แล้วสร้าง Draft Event ชื่อ 'Strategy Review Meeting' ไว้ก่อน"
```

```
"ฉันมีประชุม External Client วันพรุ่งนี้เวลา 14:00
ช่วยสร้าง Preparation Checklist โดยดูจากไฟล์ Meeting Agenda
ใน Drive และ Email Thread ที่เกี่ยวข้องกับ Client นี้ล่าสุด"
```

```
"ดูนัดหมายทั้งหมดเดือนนี้ แล้วสรุปว่า
ฉันใช้เวลาไปกับ Meeting ประเภทไหนบ้าง (Internal/External/1-on-1)
และ Meeting ไหนที่ดูเหมือนจะ Overlap หรือทำให้เหนื่อย"
```

### 📝 Meeting Notes & Follow-up (ต้องการ: Calendar + Gmail + Drive)

```
"ดูว่าประชุมอะไรเพิ่งเสร็จไปในวันนี้
แล้วช่วยร่าง Meeting Summary Template พร้อม Action Items
ให้ฉันแค่เติมรายละเอียดเอง"
```

## กลุ่มที่ 5: Slack & Team Communication

### 💬 ติดตามการสื่อสารในทีม (ต้องการ: Slack)

```
"สรุปสิ่งที่คุยกันใน #project-[ชื่อโปรเจกต์] ตลอดสัปดาห์นี้
เน้น: Decision ที่ทำไป, Issue ที่ยังค้างอยู่, และ Action Items"
```

```
"มี mention ฉัน (@username) ใน Slack ไหนบ้างที่ยังไม่ได้ตอบ
สรุปให้หน่อยว่าแต่ละอันรอคำตอบเรื่องอะไร"
```

```
"ค้นหาใน Slack ว่าเคยมีใครพูดถึงเรื่อง '[ชื่อเรื่อง]' บ้างไหม
ถ้ามี สรุปว่าตกลงกันไว้ว่าอย่างไร"
```

### 📢 ส่งรายงานและอัปเดต (ต้องการ: Slack + Drive)

```
"สร้าง Weekly Update สำหรับ Channel #general โดยใช้ข้อมูลจาก:
- Progress ของงานจากไฟล์ Project Tracker ใน Drive
- Meeting ที่ผ่านมาสัปดาห์นี้จาก Calendar
แล้วส่งไปที่ #weekly-update"
```

## กลุ่มที่ 6: Scheduled Automation (ทำงานอัตโนมัติ)

### ⏰ งานที่ทำซ้ำทุกวัน/ทุกสัปดาห์

```
[Scheduled Task — ทุกวันจันทร์ 08:30]
"ดูตาราง Meeting สัปดาห์นี้จาก Calendar
ส่ง Weekly Preview ไปที่ Slack #team-updates
รวม: Meeting ทั้งหมด, Deadline สำคัญ, สิ่งที่ต้องเตรียม"
```

```
[Scheduled Task — ทุกวันศุกร์ 17:00]
"สรุปสิ่งที่ทำได้สัปดาห์นี้จากอีเมลและ Calendar
ส่ง EOW Report ไปที่ #team-updates และ CC หัวหน้าทาง Gmail"
```

```
[Scheduled Task — ทุกวัน 07:00]
"ดูอีเมลที่มาตอนกลางคืน + ตาราง Meeting วันนี้
ส่ง Morning Brief ไปที่ Slack DM ของฉัน"
```

## 👩‍💼 สำหรับ Finance / Accounting

### งานประจำวัน

```
"ค้นหาอีเมลที่มี Attachment ชื่อ 'invoice' หรือ 'receipt' ในสัปดาห์นี้
รวบรวมและสรุป: จำนวน, ผู้ส่ง, มูลค่า, และสถานะการชำระ"
```

```
"เปิดไฟล์ 'AP_Aging_Report.xlsx' ใน Drive
สรุปรายการที่ค้างชำระเกิน 30 วัน และเกิน 60 วัน
พร้อม Highlight รายการที่มูลค่าสูงกว่า 100,000 บาท"
```

```
"ดูใบแจ้งหนี้ที่ได้รับเดือนนี้จากอีเมล
เปรียบเทียบกับ Budget ในไฟล์ Drive
แล้วบอกว่า Category ไหนที่ใช้เงินเกิน 80% ของ Budget แล้ว"
```

### Reporting & Analysis

```
"รวบรวมข้อมูลจาก Google Sheets 'Daily_Sales.xlsx'
สร้างสรุปรายงานยอดขายประจำเดือน พร้อม % เปลี่ยนแปลงเทียบเดือนที่แล้ว
ส่งไปที่ Channel #finance-report ใน Slack"
```

```
"ดู Email Thread ล่าสุดเรื่องการปิดบัญชีสิ้นเดือน
สรุป Checklist ว่ายังเหลืองานอะไรที่ต้องทำ
และแจ้งเตือน [ชื่อทีม] ผ่าน Slack"
```

### Team Oversight

```
"ดู Slack Channel #team-updates ย้อนหลัง 7 วัน
สรุปว่าแต่ละคนในทีมกำลังทำอะไรอยู่
มีใครที่ดูเหมือนติดขัดหรือต้องการความช่วยเหลือบ้าง"
```

```
"รวบรวม Status Update จาก Email ที่ทีมส่งมาสัปดาห์นี้
สรุปเป็น Executive Summary 1 หน้า
เน้น: สิ่งที่ทำเสร็จ, ปัญหาที่พบ, แผนสัปดาห์หน้า"
```

```
"ดูปฏิทินทีมใน Calendar แล้วบอกว่า
สัปดาห์หน้ามีใครที่ Over-scheduled (ประชุมเกิน 5 ชั่วโมง/วัน) บ้าง"
```

```
และมี Risk หรือปัญหาอะไรที่เกิดขึ้นระหว่างทาง"

```

## 🔧 Tips การใช้งาน MCP ให้มีประสิทธิภาพสูงสุด

### 1. ระบุช่วงเวลาให้ชัดเจน
```
❌ "ดูอีเมลเกี่ยวกับ Project X"
✅ "ดูอีเมลเกี่ยวกับ Project X ตั้งแต่ 1 พ.ค. ถึงวันนี้"
```

### 2. บอก Output ที่ต้องการ
```
❌ "สรุปอีเมล"
✅ "สรุปอีเมลเป็น Bullet Points ไม่เกิน 5 ข้อ พร้อมระบุว่าต้อง Action อะไร"
```

### 3. Multi-Step ใน Prompt เดียว
```
✅ "อ่านสรุปจากไฟล์ใน Drive → ร่างอีเมลรายงานผล → ส่งไปที่ [email]"
```

### 4. ใช้ร่วมกับ Scheduled Tasks
```
✅ ตั้ง Morning Brief ทุกวัน 08:00 โดยไม่ต้องพิมพ์ซ้ำ
✅ ตั้ง Weekly Report ทุกวันศุกร์ส่งอัตโนมัติ
```

### 5. ตรวจสอบก่อน Action สำคัญ
```
"ช่วยร่างอีเมลนี้ให้ก่อน ฉันจะ Review แล้วค่อยสั่งให้ส่ง"
(Claude จะไม่ส่งจนกว่าคุณจะยืนยัน)
```
