# คู่มือ Slack Connector — Team Communication

## การตั้งค่า
1. Settings → Connectors → Slack
2. Log in with Slack (Workspace ของบริษัท)
3. อนุมัติ Permission:
   - อ่าน Messages ✓
   - ส่ง Messages (optional — ระวัง)
   - อ่าน Channel List ✓

---

## Use Cases — Slack

### 1. Daily Summary
"สรุป Messages ใน Slack ตั้งแต่เมื่อวานเย็น
จาก Channel: #general, #sales-team, #dev-updates
บอกฉันว่ามีอะไรสำคัญที่ต้องรู้"

### 2. Find Information
"หาใน Slack ว่าเราตกลงกันเรื่อง CRM Launch Date ตอนไหน
ใน Channel #product"

### 3. Meeting Prep
"ก่อนประชุมทีมวันนี้ 14:00 น.
อ่าน Slack ทุก Channel แล้วสรุป:
- Open Items ที่ยังไม่จบ
- คำถามที่ถามค้างไว้
- Blockers ของแต่ละคน"

### 4. Auto Standup Summary
"ทุกเช้าวันจันทร์ อ่านสัปดาห์ที่แล้วใน #dev-updates
แล้วส่ง Summary ไปที่ #weekly-summary"

---

## Tips
- เริ่มจาก Read-only ก่อน
- ทดสอบใน Channel ทดสอบก่อน
- ระวัง Notification Spam
