# แบบฝึกหัด: สร้าง Budget Tracker

## Prompt

```
อ่านไฟล์ 03_budget_data.csv แล้วสร้างไฟล์ Excel ชื่อ budget_tracker.xlsx

Sheet 1 - Budget Overview:
- ตาราง Budget แต่ละ Quarter
- เพิ่มคอลัมน์ "ใช้จริง" และ "ส่วนต่าง" (ใส่ formula =Budget-ใช้จริง)
- Conditional Formatting: แดงถ้าเกินงบ, เขียวถ้าต่ำกว่างบ

Sheet 2 - Annual Chart:
- Stacked Bar Chart แสดงงบประมาณแต่ละหมวดต่อ Quarter
- Line Chart แสดงงบรวมแต่ละ Quarter

Sheet 3 - Template สำหรับ Input ใช้จริง
- เว้นช่องให้กรอก "ใช้จริง" แต่ละเดือน
```
