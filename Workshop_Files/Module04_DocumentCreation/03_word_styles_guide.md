# Word Document Style Guide

## Heading Styles
Heading 1: ชื่อเรื่องหลัก — ขนาด 18pt Bold
Heading 2: หัวข้อย่อย — ขนาด 14pt Bold
Heading 3: หัวข้อย่อยลึก — ขนาด 12pt Bold Italic

## Body Text
ฟอนต์: TH Sarabun New / Angsana New
ขนาด: 14pt (ไทย) / 11pt (อังกฤษ)
ระยะบรรทัด: 1.5

## Table Style
Header row: พื้นหลัง #003087, ตัวหนังสือขาว Bold
Alternating rows: ขาวและ Light Blue #E8F4FD
Border: 0.5pt สีเทา

"สร้าง Word Document ชื่อ [ชื่อ].docx
ใช้ Heading Styles มาตรฐาน
ตาราง: header สีน้ำเงิน ตัวหนังสือขาว
ฟอนต์: TH Sarabun New 14pt"
