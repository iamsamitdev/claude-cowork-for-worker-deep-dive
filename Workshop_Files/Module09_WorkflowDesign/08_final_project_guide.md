# Final Project: สร้าง Workflow จริงของคุณ

## โจทย์
ออกแบบและสร้าง Workflow 1 อันที่จะใช้งานจริงหลัง Workshop

## ข้อกำหนด
- ต้องเป็น Use Case จากงานจริงของคุณ
- ต้องทดสอบได้ใน Workshop วันนี้
- ต้องมีเอกสาร (Workflow Canvas) ครบ

## ขั้นตอน (45 นาที)

### Phase 1: Design (15 นาที)
1. เลือก Use Case จากงานจริง
2. กรอก Workflow Canvas
3. เขียน Prompt ร่าง

### Phase 2: Build & Test (20 นาที)
1. ทดสอบ Prompt กับไฟล์ตัวอย่าง
2. ปรับปรุงจนได้ผลที่พอใจ
3. ถ้าต้องการ: ตั้ง Scheduled Task

### Phase 3: Document & Share (10 นาที)
1. บันทึก Prompt ที่ใช้งานได้ลงในไฟล์
2. เขียน Brief Instruction สั้นๆ
3. แชร์กับเพื่อนในกลุ่ม

## Deliverable
ไฟล์: my_workflow_[ชื่อของคุณ].md ที่มี:
- Workflow Name & Description
- Trigger
- Prompt ที่ใช้งานได้จริง
- ตัวอย่าง Output
