## 🎯 หลักสูตรนี้เหมาะกับใคร

- **พนักงานออฟฟิศ (Knowledge Worker)** — จัดการเอกสาร ข้อมูล และงาน Routine
- **หัวหน้าทีม / ผู้จัดการ / ผู้บริหาร** — ยกระดับ Productivity ของทีม
- **นักบัญชี / ผู้จัดการการเงิน** — วิเคราะห์งบการเงินและสรุปรายงานด้วย AI
- **Freelancer / ที่ปรึกษา / ผู้ประกอบการ** — Scale Output โดยไม่ต้องเพิ่มคน
- **ทุกคนที่อยากใช้ AI ในงาน** — ไม่จำเป็นต้องมีพื้นฐานเขียนโปรแกรม

---

## ✨ สิ่งที่คุณจะทำได้เมื่อจบหลักสูตร

1. เข้าใจความแตกต่างระหว่าง AI Chatbot ทั่วไปกับ **Agentic AI** และใช้ Claude Cowork ได้จริง
2. อธิบายความต่างระหว่าง **Claude Cowork** และ **Claude Code** และเลือกใช้ให้เหมาะกับงาน
3. สร้างเอกสาร **Excel / Word / PowerPoint** และจัดการไฟล์ได้อย่างมีประสิทธิภาพ
4. วิเคราะห์เอกสาร งบการเงิน และสรุปรายงานธุรกิจด้วย Claude
5. เชื่อมต่อ Claude กับ **Google Drive, Gmail, Slack และ Google Calendar** ใน Workflow จริง
6. ตั้ง **Scheduled Tasks** และออกแบบ Automated Workflow ขององค์กร
7. ใช้ AI อย่างปลอดภัย เข้าใจ **Prompt Injection** และ **Data Classification**

## 💡 Tips สำคัญ 6 ข้อ

1. **เริ่มจากงาน Routine ก่อน** — งานที่ทำซ้ำทุกสัปดาห์คือ Target แรก
2. **Test Manual ก่อนเสมอ** — ก่อน Automate ต้องมั่นใจว่า Output ถูกต้อง
3. **CLAUDE.md คือสิ่งสำคัญที่สุด** — เขียนดีแล้วทุกงานจะ Output ตรงใจ
4. **Human-in-the-Loop ที่จุดสำคัญ** — AI เป็น Assistant ไม่ใช่ Decision Maker
5. **Review ทุก 30 วัน** — ตรวจสอบ Connectors, Scheduled Tasks และ Permissions
6. **ถ้าผิดพลาด ให้ Feedback** — ช่วยให้ AI ดีขึ้นสำหรับทุกคน
